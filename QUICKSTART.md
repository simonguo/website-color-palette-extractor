# 快速开始指南 🚀

## 📦 安装插件

### 方法 1：Chrome 开发者模式安装

1. **打开 Chrome 扩展管理页面**
   - 在 Chrome 地址栏输入：`chrome://extensions/`
   - 或点击菜单：更多工具 → 扩展程序

2. **启用开发者模式**
   - 在页面右上角找到"开发者模式"开关
   - 点击开启

3. **加载插件**
   - 点击"加载已解压的扩展程序"按钮
   - 选择 `website-color` 文件夹
   - 点击"选择文件夹"

4. **验证安装**
   - 扩展列表中应该出现"网站配色方案"
   - 浏览器工具栏会显示插件图标 🎨

## 🎯 快速测试

### 步骤 1：打开测试页面

```bash
# 在浏览器中打开测试页面
open test-page.html
```

或直接在浏览器中打开 `test-page.html` 文件。

### 步骤 2：激活插件

1. 点击浏览器工具栏中的插件图标
2. Chrome 侧边栏会自动打开
3. 插件自动分析页面配色

### 步骤 3：体验功能

#### ✅ Phase 2 功能测试

**颜色使用占比**
- 查看每个颜色卡片中的百分比进度条
- 占比最高的颜色会显示更长的进度条

**色彩分类**
- 查看颜色卡片上的标签：
  - 🟣 主色（紫色标签）
  - 🔴 强调（粉红色标签）
  - ⚪ 辅色（灰色标签）

**频率可视化**
- 进度条长度表示颜色使用频率
- 数字显示精确的百分比

#### ✅ Phase 3 功能测试

**1️⃣ WCAG 对比度检查**

```
1. 点击 "对比度" 按钮
2. 在"前景色"区域选择一个颜色（如深色文字）
3. 在"背景色"区域选择另一个颜色（如浅色背景）
4. 查看对比度结果：
   - 对比度比值（如 4.5:1）
   - 等级评定（AAA/AA/Fail）
   - 文本预览效果
```

**测试建议：**
- 选择深色 + 浅色：应该得到 AAA 或 AA 等级
- 选择两个浅色：可能得到 Fail 等级
- 选择两个深色：可能得到 Fail 等级

**2️⃣ 配色方案导出**

```
1. 点击 "导出" 按钮
2. 选择导出格式：
   - CSS：生成 CSS 变量
   - TW：生成 Tailwind 配置
   - FIG：生成 Figma JSON
   - JSON：原始数据
3. 自动复制到剪贴板
4. 粘贴到文本编辑器查看结果
```

**导出示例：**

CSS 格式：
```css
:root {
  --color-primary-1: #667EEA;
  --color-primary-2: #764BA2;
  --color-secondary-1: #3498DB;
  --color-accent-1: #E74C3C;
}
```

**3️⃣ 颜色搜索和过滤**

```
搜索功能：
1. 在搜索框中输入颜色值（如 #667eea）
2. 页面实时过滤匹配的颜色

过滤功能：
1. 点击 "全部" - 显示所有颜色
2. 点击 "主色" - 只显示主色
3. 点击 "辅色" - 只显示辅色
4. 点击 "强调" - 只显示强调色
```

## 🎨 常见操作

### 复制颜色值

**方法 1：概览面板**
- 点击顶部概览区的颜色块
- 自动复制颜色值

**方法 2：颜色卡片**
- 点击颜色卡片中的"复制"按钮
- 复制主颜色值

**方法 3：色阶块**
- 点击任意色阶块
- 复制该色阶的颜色值

### 刷新配色

当页面内容变化时：
```
1. 点击 "刷新配色" 按钮
2. 等待重新分析
3. 查看更新后的配色
```

### 对比度检查最佳实践

```
1. 选择主要文字颜色作为前景色
2. 选择主要背景颜色
3. 确保对比度 ≥ 4.5:1（AA 标准）
4. 大标题文字可以接受 ≥ 3:1（AA Large）
```

## 🔧 故障排除

### 问题 1：插件图标不显示

**解决方法：**
1. 检查插件是否已启用
2. 刷新 Chrome 扩展页面
3. 重新加载插件

### 问题 2：侧边栏不显示颜色

**解决方法：**
1. 点击"刷新配色"按钮
2. 检查页面是否完全加载
3. 某些网站可能阻止脚本注入

### 问题 3：导出功能不工作

**解决方法：**
1. 确保有颜色数据（至少提取到一个颜色）
2. 检查浏览器剪贴板权限
3. 手动从控制台复制数据

### 问题 4：对比度检查无响应

**解决方法：**
1. 确保选择了前景色和背景色
2. 点击颜色选择器中的颜色块
3. 查看是否有蓝色选中边框
